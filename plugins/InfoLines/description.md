InfoLines can display various information while in [2D Gamemodes](https://gimkit.wiki/wiki/Gamemodes#2D_Gamemodes). You can choose which corner of the screen it is displayed in, as well as how many decimal places should be used.

The following lines are currently available:
* Visual coordinates
* Velocity
* Physics coordinates
* FPS
* Ping
* Fish value in Fishtopia
* Plant drop rate in One Way Out
